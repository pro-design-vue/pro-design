## 特性

- 提炼自企业级中后台产品的交互语言和视觉风格。
- 开箱即用的高质量 Vue 组件。

## 安装

### 使用 npm 或 yarn 或 pnpm 安装

```bash
$ npm install ant-design-vue --save
```

```bash
$ yarn add ant-design-vue
```

```bash
$ pnpm add pro-design-vue
```
